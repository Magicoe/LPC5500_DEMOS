Overview
========

Hardware requirements
=====================
- Mini/micro USB cable
- LPCXpresso55s69 board
- 1.54inch EINK board
- Personal Computer

Board settings
==============
No special settings are required.

Prepare the Demo
================
1. Connect the EINK board with LPCXpresso55S69. 
2. Connect a micro USB cable between the PC host and the CMSIS DAP USB port (P6) on the board.
3. Download the program to the target board.
4. Either press the reset button on your board or launch the debugger in your IDE to begin running the demo.

Running the demo
================
When the demo runs successfully, you will find a picture displayed on the EINK.

Toolchain supported
===================
- Keil MDK  5.26
